# SceneWorks API

FastAPI service for versioned backend routes and all project filesystem writes.

Current routes:

- `GET /api/v1/health`
- `GET /api/v1/access`
- `POST /api/v1/auth/verify`
- `GET /api/v1/projects`
- `POST /api/v1/projects`
- `GET /api/v1/projects/{project_id}`
- `GET /api/v1/jobs/events`
